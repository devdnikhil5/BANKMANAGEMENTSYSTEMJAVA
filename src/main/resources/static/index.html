Key Features

Manager Approval Workflow

Every deposit, withdrawal, and transfer requires explicit manager approval before funds are moved.

Role-Based Access Control

Strict separation of duties: Admin manages managers, managers handle users, users transact.

Real-Time Tracking

Managers see pending approvals instantly; users track transaction statuses in their dashboard.
